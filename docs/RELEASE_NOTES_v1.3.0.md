# Claude Meter v1.3.0

The "know your context" update.

## Download

**`ClaudeMeter-portable.zip`** (attached) — unzip and run `ClaudeMeter.exe`.
No .NET installation needed. SmartScreen on first run: **More info → Run anyway**.

## New

- **Session context section** — see how full your active Claude Code session's
  context window is, with project name, model, tokens remaining until
  auto-compact, and session age. Read locally from the session transcript,
  no extra network calls. Great for knowing when to start a fresh chat before
  quality (and your token bill) suffers. Toggle via "Show session context".
- **Show limits menu** — hide any usage limit row you don't care about.
  Hidden rows still count for the tray icon and alert notifications.

## Improved

- Popup reorganized into clearly divided sections — usage limits, session
  context and session graph each get a divider and a matching small-caps
  header
